# Fullstack Authentication App

A fullstack authentication application with:

- 🔐 Google OAuth login
- 🍪 Session-based auth with secure cookies
- 🧑 User profile editing
- 💻 Frontend: React + Vite + Zustand
- ⚙️ Backend: Express + MongoDB + Mongoose
